## API Overview

TusFlow implements the TUS resumable upload protocol version 1.0.0, providing a robust and reliable way to handle file uploads.

  Base URL: `https://<your-worker>.workers.dev`
</Callout>

## Authentication

All requests must include an API key in the Authorization header:

```http
Authorization: Bearer <your-api-key>
```

## Rate Limiting

Default limits:
- 100 requests/minute per IP
- 1000 requests/minute per API key

## Protocol Support

- TUS Protocol: v1.0.0
- Extensions: creation, creation-with-upload, termination, checksum, expiration
- Maximum file size: Configurable (default 1GB)
