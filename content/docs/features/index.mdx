<Cards>
    <Card title="Resumable Uploads" href="/docs/features/resumable-uploads">
        Resumable uploads allow clients to pause and resume file uploads, making
        the process more resilient to network issues and providing a better user
        experience.
    </Card>
    <Card title="Multipart Uploads" href="/docs/features/multipart-uploads">
        Multipart uploads are a common method for uploading large files,
        allowing for parallelization and faster transfer times.
    </Card>
    <Card title="Chunk Management" href="/docs/features/chunk-management">
        Our API implements intelligent chunking to optimize file uploads based
        on network conditions and file size.
    </Card>
    <Card title="File Validation" href="/docs/features/validation">
        We support various file validation methods to ensure the integrity of
        uploaded files.
    </Card>
</Cards>
